pepino
======

El objetivo es la implementacion de una infraestructura sencilla capaz de instalarse en sistemas embebidos, pendrives booteables y computadoras de bajo rendimiento.

**Features**

- Configuracion web.
- Listen and transmit.
- Modo Relay.


**Software**

- Archlinux
- ALSA
- Icecast
- DarkIce
- Nginx
